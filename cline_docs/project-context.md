# Project Context

## What is this project?

Spark8Edge is a platform dedicated to empowering Kenya's next generation through AI and Strategic Brand Intelligence. It serves as a digital hub with the live URL: https://spark8edge.co.ke.

## Target Audience

Based on the platform's communication channels, the primary audiences are:

- Kenyan Youth (contact: youth@spark8edge.co.ke)
- Corporate Partners/Clients (contact: corporate@spark8edge.co.ke)

- **Newsletter:** Dedicated subscription handling (subscribe@spark8edge.co.ke).
- **Security:** Bot protection powered by Cloudflare Turnstile.
